# Picsearch - https://sa-picsearch.herokuapp.com/

A minimalistic web application to browse through pictures.

## Technologies

-   React.js
-   Node.js
-   Unsplash API

## Usage

1. Clone/Download this repository.
2. Run `npm start` from Command Line.
